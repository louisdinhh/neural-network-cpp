# C++ Neural Network From Scratch

A hands-on project for learning modern C++ by implementing a simple neural network from the ground up—without relying on machine learning libraries or frameworks.

This project focuses on building a solid understanding of both C++ programming principles and the core mechanics of neural networks through practical implementation.

## Learning Objectives

By completing this project, you will gain experience with:

- **C++ class design** – Organizing code into clear, reusable classes.
- **Prototyping with portable C++** – Writing standards-compliant code that builds across platforms.
- **Test early, test often** – Developing incrementally and validating functionality throughout the project.
- **Encapsulation and data hiding** – Protecting implementation details through well-designed interfaces.
- **Static class members** – Sharing state and functionality across class instances.
- **Accessor functions** – Providing controlled access to object data.
- **Const correctness** – Using `const` effectively to improve code safety and readability.
- **The `assert()` macro** – Catching programming errors during development.
- **The `std::vector` template container**, including:
  - `.size()`
  - `.back()`
  - `.push_back()`
- **Reference variables** – Passing and manipulating data efficiently without unnecessary copies.

## Project Goals

The objective is not to create a production-ready deep learning framework, but to understand how neural networks work internally while strengthening modern C++ programming skills.

Topics explored include:

- Neuron implementation
- Layer construction
- Feed-forward computation
- Error calculation
- Backpropagation
- Weight updates
- Training on sample datasets

## Why Build It From Scratch?

Using existing machine learning libraries is convenient, but implementing the algorithms yourself provides a much deeper understanding of:

- Object-oriented design
- Memory management
- Data structures
- Mathematical foundations of neural networks
- Debugging complex systems
- Incremental software development

## Prerequisites

- Basic C++ syntax
- Familiarity with functions and classes
- A C++17 (or newer) compatible compiler is recommended

## Disclaimer

This project is intended for educational purposes. The implementation prioritizes readability and learning over performance or production-level optimization.
